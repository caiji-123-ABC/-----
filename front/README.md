# 排班管理系统

基于 Vue 3 + TypeScript + Element Plus 的排班管理系统前端。

## 功能特性

- ✅ 全局规则配置
- ✅ 班次定义管理
- ✅ 组配置管理
- ✅ 人员管理
- ✅ 请假/不可用管理
- ✅ 人员锁定规则
- ✅ 调休/节假日管理
- ✅ 特殊日期规则
- ✅ 大小周配置
- ✅ 排班生成与展示
- ✅ 违规明细查看
- ✅ Excel 导出功能

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Element Plus
- Day.js
- XLSX (Excel 导出)
- Vite

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 使用说明

1. **配置全局规则**：设置连续工作天数、禁止排休星期等基础规则
2. **定义班次**：配置 A/B 班的起止时间
3. **配置组**：设置各个组的排班策略和最少在岗人数
4. **管理人员**：添加人员信息，设置所属组和班次能力
5. **管理请假**：录入人员的请假、病假、出差等不可用记录
6. **设置调休**：配置节假日和调休日期
7. **配置大小周**：设置每个月的第一周类型
8. **生成排班**：选择月份，点击生成排班
9. **查看违规**：检查排班结果中的违规情况
10. **导出数据**：将排班结果导出为 Excel 文件

## 数据存储

系统使用 localStorage 存储所有配置和排班数据，刷新页面后数据会自动恢复。

## 注意事项

- 排班生成算法为简化版本，实际使用中可能需要根据业务需求优化
- 连续工作天数校验等功能需要进一步完善
- 建议定期导出数据备份
